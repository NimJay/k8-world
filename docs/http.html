HTTP

HTTP is the language (application-level protocol) that web browsers and web servers use to communicate.

When you open example.com, your browser sends a message to example.com's server (computer) that says, "give me your home page". Then example.com's server responds with the contents of the home page. Most of the resources that you view and submit on various websites (such as images, personal data, and form submissions) are loaded in this manner. The format of these messages (requests from the browser and responses from the server) is called "Hypertext Transfer Protocol" (HTTP).
